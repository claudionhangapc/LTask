Este é um projeto que vai ser desenvolvido com 
js(vue)
html
css
php(possivelmente)
mySql(possivelment)

o projeto é uma todoList -> completa 



como fazer

	-> uma tela para entrar (login) pode ser por google ou uma outra mesmo

	-> tela principal
		
		-> Caixa de entrada ->
		-> Hoje -> (onde vai mostrar todas as atividades que foram registradas hoje)
		-> Em Breve -> vai mostrar todos os projetos que serão realizados em breve
		-> Projetos -> onde vai mostrar todos projetos da pessoa 
			- ( um projeto contem nome, cor, pode ser adicionado aos favoritos)
			- um projeto pode ter varias listas de tarefas a fazer
		-> campos para filtrar tarefas ( pedente, entregues, fazendo)



------------------------------------------------------------------------------------------------------------------

		-> uma tarefa contem:
			* titulo da tarefa 
			* projeto onde esta alocado
			* comentario
			* etapas (Pendente, Fazendo, Entregue)
			* pode se definir prioridade
			* opção para agendar
